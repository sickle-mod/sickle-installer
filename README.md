# How to download and run Sickle Mod Manager
1. Download SickleInstall.bat: https://github.com/sickle-mod/sickle-installer/blob/main/SickleInstall.bat
2. Go to your download folder and right-click the file and click Properties
3. Check "Unblock" at the bottom and click OK
4. Double-click the file. It will download and run the Sickle Mod Manager
   
# Running the Sickle Mod Manager (SMM)
1. The SMM will launch automatically. 
2. If the status says "No Mod installed" or if the Installed version is older than the latest, click Install
3. You may close the mod manager, it does not need to remain open to play Scythe
4. Play Scythe!

# Uninstall the Sickle Mod Manager (SMM)
1. Launch the SMM
2. Click Uninstall. Your original file will be restored.

# Review your game logs
1. Launch the SMM
2. click Open Logs Folder
3. Logs can be easily sorted by time.
